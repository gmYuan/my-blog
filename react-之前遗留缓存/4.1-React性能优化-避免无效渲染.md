# 【中频】React 性能优化：如何避免无效渲染？

## 常怎么问

- React 怎么避免组件重复渲染？  
- `useCallback` / `useMemo` / `memo` 分别用在什么场景？  
- 什么是「无效渲染」？怎么排查？

## 答题要点

### （暂无内容 - 后续章节会详细介绍）

本题涉及 **React 渲染机制**、**组件更新条件**（props/state 变化）、**缓存 API 使用时机**、**key 的稳定性**、**状态管理与 effect 副作用控制** 等优化手段。

PDF 原文提到的具体措施：
- 保证组件 **key 取值的稳定性**  
- 适时使用 **useCallback、useMemo、memo** 等 API  
- 掌握 React 中 **状态的管理方式**  
- 掌握 **effect 的使用时机**
