# state和props有什么区别？

## PDF原文内容：

**状态 state**：
- React UI = fn(state)
- state是变量，一般情况下，state改变之后，组件会更新

**属性 props**：
- React组件使用props来互相通信
- 每个父组件都可以提供props给它的子组件，从而将一些信息传递给它
- props是属性，用于父子通信，且props不可修改
