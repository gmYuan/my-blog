# 如何避免组件的无效渲染？

## 优化措施（来自文档）

### 1. 保证组件key取值的稳定性
- 不要使用index作为key
- 不要使用随机数作为key
- 使用稳定的唯一标识作为key

### 2. 使用缓存API
- **useCallback**: 缓存函数引用
- **useMemo**: 缓存计算结果
- **React.memo**: 缓存组件

### 3. 合理管理状态
- 状态尽量放在需要的组件中，避免提升到父组件
- 避免在父组件中定义会导致子组件重新渲染的状态
- 使用Context时注意拆分，避免不必要的更新

### 4. 正确使用Effect
- 依赖数组要完整准确
- 避免在useEffect中定义会导致重复执行的逻辑
- 清理副作用，避免内存泄漏
