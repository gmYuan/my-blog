# React的浅比较是如何实现的

## React 源码中的浅比较函数

```javascript
function shallowEqual(objA: mixed, objB: mixed): boolean {
  if (is(objA, objB)) {
    return true;
  }
  
  if (
    typeof objA !== 'object' ||
    objA === null ||
    typeof objB !== 'object' ||
    objB === null
  ) {
    return false;
  }
  
  const keysA = Object.keys(objA);
  const keysB = Object.keys(objB);
  
  if (keysA.length !== keysB.length) {
    return false;
  }
  
  // Test for A's keys different from B.
  for (let i = 0; i < keysA.length; i++) {
    const currentKey = keysA[i];
    if (
      !hasOwnProperty.call(objB, currentKey) ||
      !is(objA[currentKey], objB[currentKey])
    ) {
      return false;
    }
  }
  
  return true;
}
```

## 浅比较的逻辑

1. 首先使用 `Object.is()` 判断两个对象是否相同
2. 如果不是对象或者为 null，直接返回 false
3. 比较两个对象的 key 数量是否相同
4. 遍历对象的每个 key，使用 `Object.is()` 比较对应的值是否相同
5. 如果所有 key 对应的值都相同，返回 true，否则返回 false

## 注意

浅比较只比较对象的第一层属性，如果属性值是对象或数组，只比较引用是否相同，不会递归比较内部的值。
