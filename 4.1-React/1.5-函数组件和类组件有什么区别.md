# 函数组件和类组件有什么区别

## 定义

### 函数组件
```jsx
function Profile() {
  return (
    <img
      src="https://i.imgur.com/MK3eW3As.jpg"
      alt="Katherine Johnson"
    />
  );
}
```

### 类组件
```jsx
class Greeting extends Component {
  render() {
    return <h1>Hello, {this.props.name}!</h1>;
  }
}
```

## 说明

React 目前推荐函数组件。类组件已经归类到了 Legacy React APIs 中。
